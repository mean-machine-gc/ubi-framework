## Installation

```bash
npm install phenomena
```

## Usage

```bash
npx phenomena generate --spec ./business-specs
npx phenomena coverage --report
```

## Key Features

- **Example generation** — produces concrete examples from structured business specifications
- **Coverage analysis** — identifies gaps in example coverage against specification branches
- **Example management** — organise, curate, and version example sets
- **Pipeline integration** — output feeds directly into VibeRail's executable specification format
- **Traceability** — every example traces back to the business specification and decision branch it exercises
